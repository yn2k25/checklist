# FKR Adoption Workflow (Static GitHub Pages)

## What to upload
Upload **everything in this folder** to the root of your new GitHub repo:
- index.html
- styles.css
- script.js
- data.json
- images/ (folder)

## Screenshots
If your help blocks reference image files (example: `1-CF-in-PP.png`), put those files in the `images/` folder.

## GitHub Pages settings
Settings -> Pages:
- Source: Deploy from a branch
- Branch: main (or master) / root

Then visit:
https://<your-username>.github.io/<repo-name>/
